Solution:

In each server room an ESP8266 device with a DHT22 sensor collects and sends temperature and humidity data to AWS IoT via Wifi using MQTT. Data is forwarded to Amazon Timestream, which can be queried and presented in Amazon managed Grafana. Various alerts can be set up in Grafana to send messages to Slack and emails to admins when certain conditions are met.

Proximate price range:

AWS Costs:

Each device sends data every 5 seconds, which includes:

  • Device ID
  • Current temperature
  • Current humidity

Assuming each data point is 512 bytes:

  • Number of messages per device per month:
    • 1 message every 5 seconds
    • 60 seconds/minute * 60 minutes/hour * 24 hours/day * 30 days/month = 2,592,000 seconds/month
    • Number of messages per device: 2,592,000 / 5 = 518,400 messages/month
  • Total messages for 4 devices:
    • 518,400 messages/device * 4 devices = 2,073,600 messages/month
  • Cost Calculation for AWS IoT Core:
    • Message Cost: 2.0736 million messages * $5/1 million messages = $10.37/month
    • Rules and Actions Cost: 2.0736 million messages * $0.15/1 million messages = $0.31/month

Amazon Timestream

Storing the data for one month:

  • Data Volume:
    • Each message is 512 bytes.
    • Data per device: 518,400 messages * 512 bytes = 265,420,800 bytes ≈ 0.25 GB/month
    • Total data for 4 devices: 0.25 GB/device * 4 devices = 1 GB/month
  • Cost Calculation for Amazon Timestream:
    • Ingested Data Cost: 1 GB * $0.50/GB = $0.50/month
    • Storage Cost: 1 GB * $0.03/GB = $0.03/month
    • Query Cost: Assuming 1 million rows scanned per month = $0.01

Amazon S3 (for backup)

Assuming the data is moved to S3 after one month:

  • Data Volume: 1 GB/month (same as Timestream storage)
  • Storage Cost:
    • S3 Standard Storage: $0.023 per GB
  • Cost Calculation for Amazon S3:
    • Storage Cost: 1 GB * $0.023/GB = $0.023/month

Amazon Managed Grafana

  • Grafana Cost: $9/month for 1 user

Total Monthly Cost for AWS

Adding up the individual costs:

  1. AWS IoT Core:
    • Message Cost: $10.37
    • Rules and Actions Cost: $0.31
    • Total: $10.68/month
  1. Amazon Timestream:
    • Ingested Data Cost: $0.50
    • Storage Cost: $0.03
    • Query Cost: $0.01
    • Total: $0.54/month
  1. Amazon S3:
    • Storage Cost: $0.023/month
  1. Amazon Managed Grafana:
    • Total: $9/month

Total Estimated Monthly Cost: $10.68 (IoT Core) + $0.54 (Timestream) + $0.023 (S3) + $9 (Grafana) = $20.243/month which is around 7,300 HUF.

Hardware Costs:

Prices per installation is approximately 6,300 HUF, for four server rooms it comes to 25,200 HUF.

Initial total monthly cost:

Assuming these characteristics initial total monthly comes to 32,500 HUF.



This number can be reduced by:

going on-prem

using a micro USB to supply power to the ESP8266 instead of the PSU

sending and storing fewer characters, less data

sending data less frequently

setting up a lifecycle to erase (or store in S3) old data
